Main.java

Entry point for the Search Optimize application.

This program loads a list of file record entries into memory and provides a simple command-line interface that allows the user to:

  • add new records
  • delete existing records
  • search for records containing a specific substring
  • print all stored records

Commands are read from standard input and processed continuously until the user enters "exit".


import java.io.IOException;
import java.util.Scanner;

public class Main {

    public static void main(String[] args) throws IOException {
        System.out.println("Welcome to the Search Optimize");

        Records records = new Records();
        records.readFilesToAlist();

        Scanner scanner = new Scanner(System.in);

        System.out.println("Commands:\n" +
                "1.update <fileName>\n" +
                "2.delete <fileName>\n" +
                "3.search <fileName>\n" +
                "4.print\n" +
                "5.exit");

        while (true) {

            System.out.print("\n> ");
            String line = scanner.nextLine().trim();

            if (line.equals("exit")) break;

            String[] parts = line.split("\\s+", 2);
            String command = parts[0];
            String value = parts.length > 1 ? parts[1] : "";

            switch (command) {
                case "update":
                    records.updateList(value);
                    break;

                case "delete":
                    records.deleteFromList(value);
                    break;

                case "search":
                    records.search(value);
                    break;

                case "print":
                    records.printList();
                    break;

                default:
                    System.out.println("Unknown command: " + command);
            }
        }

        System.out.println("Goodbye.");
    }
}

Records.java

Manages an in-memory collection of file record entries.

This class loads file names from a text file into a list, allows searching within the loaded records, and supports basic operations such as:

  • adding records
  • removing records
  • printing records

Each file name is stored as a separate record in memory.


import java.io.*;
import java.util.LinkedList;
import java.util.List;

public class Records {

    private File file = new File("src/2000_file_records.txt");

    private List<String> list = new LinkedList<>();

    public void readFilesToAlist() throws IOException {

        BufferedReader reader = new BufferedReader(new FileReader(file));
        String line;

        while ((line = reader.readLine()) != null){
            list.add(line);
        }
    }

    public List<String> search(String word) {

        System.out.println("Searching for: " + word + "\n");

        long startTime = System.currentTimeMillis();

        List<String> newList = list.stream()
                .filter(item -> item.contains(word))
                .toList();

        long endTime = System.currentTimeMillis();
        double elapsedSeconds = (endTime - startTime) / 1000.0;

        System.out.println("Time elapsed: " + elapsedSeconds + " seconds\n");
        newList.forEach(System.out::println);

        return newList;
    }

    public void updateList(String newFile){
        list.add(newFile);
    }

    public void deleteFromList(String fileToBeDeleted){
        list.remove(fileToBeDeleted);
    }

    public void printList(){
        list.forEach(System.out::println);
    }
}
